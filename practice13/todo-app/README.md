## Практическое занятие 13: Создание простого приложения на React

## Что сделано
1. Создано React-приложение «Менеджер задач» (Todo Manager) с использованием Vite.
2. Реализованы компоненты: AddTodoForm (добавление задач),TodoItem (отображение и управление задачей), TodoFilters (фильтрация по статусу).
3. Добавлена функциональность: добавление задач, отметка выполнения, удаление, фильтрация (все/активные/выполненные), счётчик оставшихся задач.
4. Реализовано сохранение задач в localStorage для их восстановления после перезагрузки страницы.
5. Выполнена настройка и публикация проекта на GitHub Pages.
